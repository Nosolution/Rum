# Rum

希望通过本项目的实验找到一种合适的web项目开发模板，以备日后使用。

## 计划

### 环节与使用工具

- 开发：探索合理的项目架构，有强大的业务表达能力且不冗余，围绕Spring Boot，Spring Security，Spring Cloud等环Spring生态圈
- 测试：找到完备测试的范式，使用多种测试工具
- 部署：使用Jenkins，docker等
- 文档：追求便捷规范的文档管理方式，比如使用api管理工具如eolinker等

### 常见的需求的实现

- 路由鉴权
- 用户模型
- 数据库的使用，包括设计与性能考虑等
- 组队
- 帖子
- 文件上传下载
- 关注
- 讨论组
- 私信
- 评分
- 用户等级
